# Fake Review Detection System

## Overview
Enterprise-scale fake review detection system leveraging Graph Neural Networks, NLP embeddings, and reviewer-graph analytics to identify coordinated review fraud in e-commerce ecosystems.

## Features
- Coordinated Fraud Detection
- Graph Network Analytics
- NLP Embedding Analysis
- Pre-trained GNN Model

## Tech Stack
- Python
- PyTorch
- Graph Neural Networks(GNN)
- Streamlit

## ML Concepts Used
- Graph Neural Networks
- Natural Language Processing(NLP)
- Text Embeddings
- Node Classification

## Project Structure
app.py train_gnn.py utils.py fraud_gnn.py reviews.csv requirements.txt
